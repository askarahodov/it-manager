# IT Manager — Product

## Vision
IT Manager — единая платформа для инвентаризации, безопасного доступа и автоматизации инфраструктуры, которая уменьшает операционные риски и время реакции на инциденты.

## ICP / Пользователи
- IT/DevOps/SRE команды (5–50 инженеров)
- Системные администраторы в компаниях с гибридной инфраструктурой
- Инфраструктурные команды в SMB/Enterprise, где нужен аудит действий и безопасное управление доступом

## Основные сценарии (MVP+)
- Инвентаризация хостов и групп, мониторинг статусов
- Безопасное хранение и ротация секретов
- Управление доступом (RBAC, approvals)
- Автоматизация через playbooks + расписания + триггеры
- Аудит действий и уведомления

## Ценность
- Снижение риска “ручных” действий
- Прозрачность и аудит для compliance
- Быстрый time‑to‑remediation за счёт автоматизации

## Метрики продукта
- MTTR (mean time to remediate)
- Доля автоматизированных операций
- Количество отклонённых/подтверждённых approvals
- Количество инцидентов, связанных с доступом
- Уровень coverage (кол-во хостов/групп под управлением)

## Принципы
- Безопасность по умолчанию
- Минимальный риск “секретов на диске”
- Ясные, объяснимые действия и аудит
- Архитектура, готовая к масштабированию
